# fukidashi

*fukidashi* (吹き出し) — the speech bubble in manga.

A tool that reads manga the way a translator does: holistically and page by page, so translations carry the accuracy of full context — not just isolated bubbles.

## Why

Most manga translation tools OCR each speech bubble in isolation and translate it blind. That loses everything that makes a translation accurate: who is speaking, what happened on the previous page, tone, honorifics, running jokes, and the visual context of the panel itself.

## How it works

1. **Read the whole work first** — build context across chapters and pages: characters, relationships, tone, recurring terms.
2. **Then translate page by page** — each page is translated with the full story context in hand, panel order respected, speaker attribution intact.
